## Installation

1. Install node.js
2. Install IDE
3. Install Android Studio (Emulator)
4. Install Appium
5. Install webdriver.io

## How to run

API :
- Run command > npx mocha [path file]

Mobile :
- Open Emulator
- Run Appium
- Run command > npx wdio
